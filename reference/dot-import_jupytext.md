# Import the jupytext Python library

Load the jupytext Python library in an existing basilisk process.

## Usage

``` r
.import_jupytext(proc)
```

## Arguments

- proc:

  A process object generated by
  [`basilisk::basiliskStart()`](https://rdrr.io/pkg/basilisk/man/basiliskStart.html)

## Value

`NULL`, invisibly. The jupytext module is stored within the `proc`
process object and can be passed to subsequent `basiliskRun()` calls by
setting `persist = TRUE`, passing a `store` argument to `fun` and
accesing the module with `store$jupytext`.
